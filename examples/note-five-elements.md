# 示例：一篇完整笔记的写法（五元组）

> 演示用，内容为通用占位主题。封面用下方 YAML 跑 `render_cover.py`，正文复制到小红书。

## 封面（保存为 cover.md 后渲染）

```yaml
---
theme: green
tag: 实操干货
eyebrow: 房间总是乱？
title: 学会这**5个**\n收纳习惯
subtitle: 不用买收纳盒，\n**今天就能上手**
chips: ["# 收纳", "# 居家", "# 生活技巧"]
author: 你的昵称
author_sub: 一句话介绍你自己
---
```

```bash
python scripts/render_cover.py cover.md   # 生成 cover.png
```

## 正文（直接复制到小红书）

**标题**：学会这5个收纳习惯，房间再也乱不了

**开头钩子**：
房间总是收完两天又乱？
不是你懒，是少了几个「随手习惯」👇

**正文**（观点→证据→延展）：
1️⃣ 物归原位的前提是「每样东西有固定的家」
没有固定位置，再勤快也会乱。先给高频物品定位。

2️⃣ 用「进一出一」控制总量
买一件新的，就处理掉一件旧的，源头不堆积。

3️⃣ 台面只留每天都用的
其余收进抽屉，视觉清爽 80% 靠台面。

4️⃣ 睡前 5 分钟「快速复位」
不打扫，只把东西放回家，第二天清清爽爽。

5️⃣ 周末做一次「15 分钟微整理」
小步高频，比攒到大扫除轻松得多。
